# 13 — Pronunciation Guide for Sanskrit (for English Speakers)

A practical, non-technical guide. The goal: a foreign reader can *say* the words well enough to recognize and
respect them — not flawless Sanskrit scholarship. Keep it light and friendly.

## 1. How Sanskrit is presented in this book
Every Sanskrit term appears up to three ways on first use:
- **Devanagari** — the native script (देवनागरी).
- **IAST** — the International Alphabet of Sanskrit Transliteration (diacritical roman script), e.g.,
  *bhagavad-gītā*.
- **plain meaning** in English.

## 2. The sounds (with a "say it like" cue)

| Letter / diacritic | Sounds like | Example |
|---|---|---|
| **a** | the *u* in "but" (short) | *dharma* → DHUR-mə |
| **ā** (long a) | the *a* in "father", held longer | *gītā* → GEE-taa |
| **i** | the *i* in "sit" | *iti* → IT-i |
| **ī** | the *ee* in "see" | *gītā* → GEE-taa |
| **u** | the *u* in "put" | *guna* |
| **ū** | the *oo* in "food" | *mūla* |
| **ṛ/ṝ** | a rolled *ri* (like "rri" in "risk") | *kṛṣṇa* → KRISH-na |
| **e** | the *ay* in "say" (always long) | *deva* → DAY-va |
| **o** | the *oe* in "go" (always long) | *om* → OHM |
| **ai** | the *i* in "ice" | *vaiṣṇava* |
| **au** | the *ou* in "house" | *gauḥ* (cow) |

### Consonants with dots/lines (retroflex & aspirated)
| Letter | Sounds like | Note |
|---|---|---|
| **ṭ, ṭh, ḍ, ḍh, ṇ** | a "thicker" t/d/n, tongue curling back | between English t/d and the American "r"-colored sound |
| **k, kh, g, gh** | as in English, with a puff of air after *h* | *kh* ≠ German *ch*; it's k+h |
| **ch, jh** | as in "church"/j+h | |
| **ṭh/gh** etc. | aspirated (follow with a sharp breath) | never silent, never like English "th" |
| **ṅ / ñ** | the *n* in "sing" / "canyon" | |
| **h** | a soft h, plus a faint echo of the preceding vowel | *buddha* |

### Key consonants learners get wrong
| Letter | Sounds like | Example |
|---|---|---|
| **ś / ṣ / s** | all roughly "sh"-ish; *ś/ṣ* = *sh* in "ship"; *s* = plain *s* | *viṣṇu* → VISH-nu; *krishna* |
| **v** | between English *v* and *w* | *veda* → VAY-da (closer to *way*-da) |
| **c** | always *ch* as in "church" — never *k* | *cakra* → CHAK-ra |
| **j** | as in "jam" | |
| **y** | as in "yes" — never a vowel | |
| **m / ṃ (anusvāra) / ḥ (visarga)** | *ṃ* = nasal hum before next consonant; *ḥ* = a soft *h* with an echo vowel | *aḥ* → "aha" |

## 3. Stress / accent (simplified)
- In IAST there are no stress marks; Sanskrit was chanted, not stress-accented like English.
- Practical rule for English speakers: **give a slight weight to the long vowel** in a word, and pronounce
  every syllable.
- The Gita is traditionally **recited/sung** (*gītā* = "the sung one"); encourage readers to *listen* to a
  recitation — hearing it conveys what any guide cannot.

## 4. Words you'll meet most (pronounced)
- **Bhagavad Gītā** → bhug-uh-vud GEE-taa ("Song of the Divine")
- **Kṛṣṇa** → KRISH-na
- **Arjuna** → ur-JOO-na
- **Dharma** → DHUR-ma
- **Karma** → KUR-ma
- **Ātman** → AAT-mun
- **Brahman** → BRUH-mun (note: NOT *Brahmin*, which is a person)
- **Mokṣa** → MOK-sha
- **Yoga** → YOH-ga
- **Bhakti** → BHUK-ti
- **Guṇa** → GOO-na
- **Sattva / Rajas / Tamas** → SUT-va / RUJ-us / TUM-us
- **Sannyāsa** → sun-YAA-sa
- **Ahiṃsā** → uh-HIM-saa (non-violence)
- **Oṃ (ॐ)** → a deep, sustained "A-U-M"

## 5. A gentle encouragement for the reader
Sanskrit's beauty is in its sound; mispronouncing a word in this book is not a sin. The meaning is what
matters. When in doubt, say it slowly, and listen to a traditional chanting of the Gita to feel its rhythm.
