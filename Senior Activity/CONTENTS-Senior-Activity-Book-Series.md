**Purpose:** the canonical breakdown of *what is inside each book* — every activity type, by difficulty tier, with exact counts, grid specs, layout, page budget, and cognitive-domain mapping. This is the single source of truth for the puzzle mix. The `PLAN-` and `RESEARCH-` files reference it.

## 1. Scope — identical structure across all 10 books
Every book in the *Bright Mind Large-Print Puzzles & Brain Games* series shares the **exact same activity structure, counts, difficulty distribution, and page budget.** Only the **theme content** differs per book:
- Word-search word banks, word-scramble words, trivia questions, and maze names are themed.
- The single **Home Remedies & Wellness Wisdom** page is themed where natural (Food → kitchen-spice remedies; Faith → mindful practices; Outdoors → herbs).

Because the structure is fixed, this document describes **one book** — multiply by 10 for the full series.

## 2. Difficulty framework (3 tiers, all senior-appropriate)
All three tiers are designed for **active seniors / brain-health** solvers. There is deliberately **no "Hard / Expert / Insane" tier** — the hardest level here is a *gentle stretch*, never frustrating. Difficulty is tuned via grid size, word direction, hint presence, and number of givens.

| Tier | Label | Target solver feel | Dial settings (typical) |
|---|---|---|---|
| **E** | Easy / Warm-up | Confidence-building, relaxing, "I can do this." | Smallest legible grids; words across/down only; hints given; many givens. |
| **M** | Medium / Steady | The core "brain exercise" band — engaged but comfortable. | Standard grids; diagonals allowed; category hint; moderate givens. |
| **C** | Challenger | A gentle stretch for experienced solvers; still fully solvable. | Largest grids; diagonals + reversed words; no hint; fewer givens; unique solution guaranteed. |

**Accessibility (all tiers):** body text **18 pt** (Atkinson Hyperlegible; floor 16 pt), grid letters 20–22 pt, true black on cream, one word-search or maze per page, 2 sudoku/scramble/trivia per page, generous gutter. Full geometry/font spec in `LAYOUT-`.

### 4.6 Bonus Coloring — 4 pages
- Simple, bold line art, themed (e.g. vintage car silhouette, bird, teapot).
- No difficulty tier — therapeutic/relaxation value; large open shapes for easy shading.

### 4.7 Home Remedies & Wellness Wisdom — 1 page
- Safe, time-tested folk/traditional remedies, themed where natural.
- **Hard constraint:** the word "Ayurveda" must NOT appear anywhere.
- **Mandatory disclaimer on the page:** "Traditional home remedies shared for cultural enjoyment and general wellbeing. This is not medical advice; consult a healthcare professional." Folk-wisdom framing; never prescriptive; never dosing-based.

## 5. Cognitive-domain mapping (why this mix)
Variety is the differentiator and the "brain exercise" hook. The mix deliberately spans cognitive domains:

| Activity | Primary domain(s) |
|---|---|
| Word Search | Visual scanning, attention, language |
| Sudoku | Logic, working memory, number reasoning |
| Word Scramble | Language, lexical recall, pattern |
| Trivia / Finish-the-Phrase | Long-term memory, semantic recall |
| Mazes | Visual-spatial planning, executive function |
| Coloring | Fine motor, relaxation, creativity |
| Home Remedies | Wellbeing, engagement, nostalgia |

- Font: body ≥ 18 pt (hard floor 16 pt); grid letters ≥ 20 pt; margins/bleed/gutter conform to KDP template.
