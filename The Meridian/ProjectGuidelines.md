# THE MOVING TARGET — Project Guidelines

Working rules for this book. Read before any drafting, reviewing, or revising.

---

## 1. THE NON-NEGOTIABLE REVIEW RULE

> Never ask an AI / reviewer / MCP to give its view, review, or judgement. Always ask it to **describe in extreme detail what it is seeing**.

When soliciting any feedback, prompts must request **concrete description** (what is on the page, what is repeated, what a character does/says, what a reader's eye is drawn to, where tension rises or falls, what is planted and when it pays off) — **not opinions, scores, or verdicts**. Scores live only in `Ratings.md` and are the author's to assign after reading the descriptions.

---

## 2. CRAFT STANDARDS (from the development plan §1)

- **Open-loop stacking:** never close a question without opening two.
- **Every chapter ends on a hook** — varied (revelation, question, foreboding, image), not always a cliffhanger.
- **Short chapters:** ~1,800–2,200 words. ~88,000 words total / ~40 chapters + prologue + epilogue.
- **A visible ticking clock:** the 14-hour run. Every chapter header carries a **station timestamp** (countdown).
- **One escalating set-piece per act:** at-speed boarding · vault-car crack · at-speed escape / car-to-car hunt.
- **Fair-play misdirection:** every twist foreshadowed ≥2–3× (tracked in `TWIST_MAP.md` Foreshadowing Ledger). No cheating.
- **The plan you're shown isn't the plan:** audience plan → contingency → true objective. Execution deviates from the briefing.
- **Competence porn:** experts being brilliant under pressure; every skill paired with a flaw + secret + betrayal seed.
- **POV:** tight 3rd-person, rotating crew POVs, cinematic and objective-ish — **never** unreliable first-person (that is *The Fourth Step*'s DNA and is off-limits here).

---

## 3. CONTENT GUARDRAILS (from the plan §8) — enforced in every scene

- **Clean:** no erotica; violence **implied, not graphic**; no gore; **minimal swearing** (mild only). Reads for **14+**.
- **No Indian characters** (per author request).
- **Global/international cast** (US, UK, French-Moroccan, Korean, Brazilian, Nigerian, Italian, Serbian, Swiss/German, etc.).
- **Translation-friendly:** avoid puns, idioms, and wordplay that break across languages. (The puzzle layer is designed to survive translation — see `PUZZLE_LAYER.md` §9.)
- Wit/banter to relieve tension; **real emotional stakes** (grief, identity, justice, found-family).
- No scenes that depend on graphic cruelty; menace is conveyed through competence, control, and consequence, not blood.

---

## 4. DISTINCTIVENESS FROM *THE FOURTH STEP* (catalogue diversity)

This book must NOT repeat *The Fourth Step*'s DNA. Quick reference:

| Axis | The Fourth Step (existing) | THIS book |
|---|---|---|
| POV | Multi-1st-person, contradictory | Tight 3rd-person, rotating crew POVs |
| Engine | Psychological dread / memory | Plot mechanics / the heist as a machine |
| Voice | Literary, introspective | Propulsive, witty, "movie-on-the-page" |
| Setting | One claustrophobic house | A single night on a never-stopping train |
| Tension | "What is real?" | "What's the real plan? / Will it work?" |
| Twist type | Memory unreliability | Misdirection of competence (the con) |

---

## 5. PIPELINE (mirrors *The Fourth Step*)

Bible + Twist Map + Puzzle design + Beat Sheet → **Draft 1** → multi-model AI reviews (`Reviews/`, describe-don't-judge prompts) → **Draft 2** → … → rescore `Ratings.md` (grand average target **8.5+**) → `BETA_READER_GUIDE.md` → copy edit → build EPUB (`build_epub.py` etc., later) → translations (FR/DE/ES, later).

**Update convention:** date-stamp every draft's scores in the `Ratings.md` update log. Track foreshadowing payoffs in the `TWIST_MAP.md` ledger as they are written.
